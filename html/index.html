Basic graphs are supported straight out of the box, ensuring the ability to deliver data in a clearly familiar format to those who may not be most up-to-date with the latest graphing techniques.

Line Graph Bar Graph Pie Chart 
var ctx1 = $("#lineGraph")[0].getContext('2d');
var myChart1 = new Chart(ctx1, {
  type: 'line',
  data: {
    labels: ['M', 'T', 'W', 'T', 'F', 'S', 'S'],
    datasets: [{
      label: 'apples',
      data: [12, 19, 3, 17, 6, 3, 7],
      backgroundColor: "rgba(153,255,51,0.4)"
    }, {
      label: 'oranges',
      data: [2, 29, 5, 5, 2, 3, 10],
      backgroundColor: "rgba(255,153,0,0.4)"
    }]
  }
});

Along with basic graphs, ChartJS also supports more advanced graphs. Namely: Polar, Radar, and Doughnut.
